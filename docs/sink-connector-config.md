`encoding`  
The encoding schema for the audio.

-   Type: string
-   Default: wav
-   Importance: high

`print.stderr`  
Whether to print the stderr for the invoked process to the console

-   Type: boolean
-   Default: false
-   Importance: low

`print.stdout`  
Whether to print the stdout for the invoked process to the console

-   Type: boolean
-   Default: false
-   Importance: low
