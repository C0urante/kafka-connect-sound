`topic`  
The Kafka topic to write to.

-   Type: string
-   Importance: high

`encoding`  
The encoding schema for the audio.

-   Type: string
-   Default: wav
-   Importance: high

`print.stderr`  
Whether to print the stderr for the invoked process to the console

-   Type: boolean
-   Default: false
-   Importance: low
